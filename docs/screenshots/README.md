# Screenshots

Public-facing screenshots of Maple for README, dev.to posts, release notes, etc.

Conventions:
- PNG, captured in macOS dark mode, ideally with the window hero-sized (~1400×900).
- Name by feature: `commit-graph.png`, `conflict-resolution.png`, `diff-blame.png`, etc.
- Compress with [TinyPNG](https://tinypng.com) before committing.
- Avoid anything identifying a private repository or colleague.

Hosting: these files are referenced by raw GitHub URL, e.g.
`https://raw.githubusercontent.com/poolcamacho/Maple/master/docs/screenshots/<name>.png`
